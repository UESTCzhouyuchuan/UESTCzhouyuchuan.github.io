              常见的red white gray

              # 特殊关键字

              # currentColor

              currentColor 代表当前元素设置的color值,或者最近的祖先元素设置的color值。 例如:

              <div style="color:green; border: 1px dashed currentColor;">

              # rebeccapurple

              颜色 rebeccapurple 等价于颜色 #639。

              # RGB

              • R 为 red
              • G 为 green
              • B 为 black
              color: rgb(255,255,255);
              1

              光的三原色红绿蓝。

              # 十六进制颜色

              # 6 位表示法。

              前两个值为R值，中间两个值为G值，后两个值为B 值。即对应RGB

              比如

              • #FFFFFF可简写为#FFF。
              • #000000简写为#000。

              缩写的格式: #663399 === #639，即 6 个 16 进制简写成 3 个，实际值是每个值的双拼写的组合。

              # 8 位表示法。

              多出来的最后两位表示A透明度。对应RGBA

              # RGBA

              A 是 alpha 即透明度，值在 0-1，0 为完全透明，1 不透明。

              color: rgba(0,0,0,0.5);
              1

              # HLS

              background-color: hsl(30,100%，50%);
              1

              # H

              H 为 hue，即色相，色相即颜色值。 取值范围是 0-360

              • 0 和 360 是红色
              • 30 是橙色
              • 180 表示绿色
              • 240 表示蓝色

              # S

              S 为 saturation,饱和度。表示色彩鲜艳程度,0%-100%,饱和度越高，颜色越鲜艳。

              # L

              L 为 lightness,亮度。表示色彩明暗程度,0%-100%,值越高，颜色越亮。

              # HSLA

              和 RGBA 类似，A 是 alpha 即透明度，值在 0-1，0 为完全透明，1 不透明。

              background-color: hsla(30,100%,50%,.5);
              1
